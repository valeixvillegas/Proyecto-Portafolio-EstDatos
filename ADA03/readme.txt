Equipo: E01

Descripción:
Este programa implementa una Lista Doblemente Ligada en C
para administrar el contenido del archivo Movies.csv.
Cada nodo de la lista almacena una línea completa del CSV,
conservando todas las columnas y filas del archivo.

---------------------------------------------------
                FUNCIONALIDADES
---------------------------------------------------

1. Cargar los datos desde Movies.csv a la lista.
2. Mostrar todas las líneas del archivo.
3. Buscar información de las peliculas según su id.
4. Insertar una nueva línea (película).
5. Eliminar una línea (con confirmación).
6. Actualizar una línea existente.
7. Salir del programa.

---------------------------------------------------
                INSTRUCCIONES
---------------------------------------------------

1. Asegúrate de tener los siguientes archivos:
   - main.c
   - Movies.csv

2. Compilación:
   gcc main.c -o movies

3. Ejecución:
   ./movies

4. El archivo Movies.csv debe estar en el mismo directorio
   que el ejecutable.
