# Proyecto-Portafolio-EstDatos
Este Portafolio de Evidencias contiene todas las actividades de aprendizaje realizadas en el curso de la asignatura Estructuras de Datos ago-dic 2025.

## Listado de Actividades de Aprendizaje

# [ADA01 - Pilas]
> **Descripción:** [El objetivo fue convertir expresiones en notación infija a postfija y luego evaluar la expresión resultante, respetando la prioridad de operadores.]
[Enlace a la carpeta ADA01](./ADA01)

# [ADA02 - Colas]
> **Descripción:** [El programa debía generar clientes aleatorios, asignarlos a colas y calcular estadísticas de eficiencia, como el tiempo promedio de espera]
[Enlace a la carpeta ADA02](./ADA02)

# [ADA03 - Listas]
> **Descripción:** [Se requería realizar Búsqueda (por título o actor), Inserción, Eliminación y Actualización de los datos de las películas.]
[Enlace a la carpeta ADA03](./ADA03)

# [ADA04 - Árboles]
> **Descripción:** [Creación de un Sistema de Estudiantes donde los datos se ordenaban automáticamente por la matrícula. Las funciones clave eran la Inserción, Eliminación y la generación de Reportes ordenados]
[Enlace a la carpeta ADA04](./ADA04)

# [ADA06 - ImpOrdenamiento]
> **Descripción:** [El objetivo fue utilizar un repositorio de Git para implementar un algoritmo específico, y practicar el flujo de trabajo profesional (clonar, crear ramas, commits claros y Pull Requests) para integrar el código con el trabajo de otros equipos.]
[Enlace a la carpeta ADA05](./ADA06)

# [ADA07 - Hashing]
> **Descripción:** [Implementación de la Tabla Hash con dos métodos de manejo de colisiones, Registro de Estudiantes por matrícula (clave numérica) usando Direccionamiento Abierto (sondeo) yTabla de Símbolos (en un compilador) usando nombres (string) como clave y Encadenamiento Separado (Listas Ligadas) para colisiones]
