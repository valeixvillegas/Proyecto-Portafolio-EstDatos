El ADA requería el desarrollo de dos aplicaciones diferentes, cada una utilizando un método distinto de manejo de colisiones, lo cual es fundamental en el estudio de las Tablas Hash.
Este ADA fue uno de los más reveladores porque me mostró que la búsqueda ultrarrápida que proporciona una Tabla Hash depende enteramente de cómo se manejan los problemas (las colisiones).
Al tener que implementar dos aplicaciones con métodos de colisión diferentes, comprendi sus ventajas y desventajas.
El mayor reto conceptual fue darme cuenta de que una mala función hash puede arruinar toda la tabla, generando demasiadas colisiones y degradando el rendimiento a una búsqueda.
